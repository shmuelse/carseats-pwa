# 🚗 מושבים — Car Seats Manager

אפליקציית PWA לניהול חכם של מושבי ילדים ברכב משפחתי.  
עובדת ללא אינטרנט, כל הנתונים נשמרים על המכשיר בלבד.

**🔗 [פתח את האפליקציה](https://shmuelse.github.io/carseats-pwa/)**

---

## ✨ תכונות

- **סידור מושבים אוטומטי** — האפליקציה מחשבת מי הבא בתור לכסא הקדמי או לשורה האחורית, בהתאם להיסטוריית הנסיעות
- **נסיעות קצרות וארוכות** — תור נפרד לכל סוג נסיעה
- **ניהול הורים** — בחירה אם נוסע הורה 1, הורה 2, שניהם, או אף אחד (כשנוסע הורה אחד בלבד — ילד זוכה בכסא הקדמי)
- **ניהול ילדים** — הוספה, עריכת שם, בחירת מגדר (👦/👧/🧒), מחיקה
- **היסטוריה וסטטיסטיקות** — מי ישב הכי הרבה פעמים בכסא הקדמי
- **עיצוב מודרני** — dark mode, רספונסיבי למובייל

---

## 📱 התקנה על אנדרואיד

1. פתח את [הקישור](https://shmuelse.github.io/carseats-pwa/) ב-Chrome
2. לחץ על **"התקן כאפליקציה"** בתוך האפליקציה, או:  
   שלוש נקודות ⋮ ← "הוסף למסך הבית"
3. האפליקציה תופיע עם אייקון על מסך הבית ותעבוד גם ללא אינטרנט

---

## 🔒 פרטיות

כל הנתונים נשמרים **על המכשיר בלבד** באמצעות `localStorage`.  
אין שרתים, אין ענן, אין מעקב.

---

## 🛠️ טכנולוגיות

- HTML / CSS / JavaScript — ללא frameworks
- PWA עם Service Worker לתמיכה offline
- `localStorage` לשמירת נתונים מקומית
- מאוחסן על GitHub Pages
