# 🚗 מושבים — Car Seats Manager

אפליקציית PWA לניהול הוגן של מושבי ילדים ברכב.  
האפליקציה תומכת בשני סוגי רכב — רכב משפחתי 7 מקומות ורכב רגיל 5 מקומות — עם תורים, היסטוריה והגדרות נפרדות לפי סוג הרכב.

**🔗 [פתח את האפליקציה](https://shmuelse.github.io/carseats-pwa/)**

---

## ✨ תכונות מרכזיות

- **בחירת סוג רכב** — רכב 7 מקומות או רכב 5 מקומות.
- **זכירת הרכב האחרון** — פתיחה רגילה של האפליקציה מחזירה לרכב שנבחר בפעם הקודמת.
- **חזרה למסך בחירת רכב** — אפשר בכל זמן לבחור רכב אחר.
- **שפה** — בחירה בין עברית לאנגלית.
- **מצב תצוגה** — בחירה בין מצב כהה לבין מצב רגיל.
- **סידור מושבים אוטומטי** — האפליקציה מחשבת מי הבא בתור למושב הקדמי או לשורה האחורית לפי היסטוריית הנסיעות.
- **תורים נפרדים** — ברכב 7 מקומות יש תור נפרד לנסיעות קצרות/ארוכות ולשורה האחורית; ברכב 5 מקומות יש תור נפרד למושב הקדמי.
- **מקומות קבועים** — ניתן להגדיר לילדים מושבים קבועים. ההגדרות של רכב 5 מקומות נפרדות מההגדרות של רכב 7 מקומות.
- **ניהול הורים** — ברכב 7 מקומות בוחרים מי מההורים נוסע; ברכב 5 מקומות בוחרים הורה אחד או שני הורים.
- **ניהול ילדים** — הוספה, עריכת שם, בחירת מגדר, מחיקה והצגת סימון מקום קבוע.
- **היסטוריה וסטטיסטיקות** — שמירת נסיעות אחרונות וחישוב תורים לפי שימוש בפועל.
- **גיבוי וייבוא** — ייצוא גיבוי JSON וייבוא נתונים קיימים.
- **התקנה כאפליקציה** — כפתור התקנה שמופיע כשהדפדפן מאפשר התקנת PWA, ונעלם לאחר ההתקנה.
- **עבודה Offline** — האפליקציה נטענת ועובדת גם ללא אינטרנט לאחר טעינה ראשונה.

## 🚙 סוגי הרכב הנתמכים

### רכב 7 מקומות

| שורה | מקומות |
|------|--------|
| קדמי | נהג + נוסע אחד |
| אמצע | 3 מקומות |
| אחורי | 2 מקומות |

ברכב 7 מקומות:

- אפשר לבחור נסיעה קצרה או ארוכה.
- יש תורים נפרדים לפי סוג נסיעה.
- ילד יכול לשבת מקדימה רק כאשר נבחר הורה אחד בדיוק.
- ניתן להגדיר מושב קבוע מתוך כל מושבי הרכב.

### רכב 5 מקומות

| שורה | מקומות |
|------|--------|
| קדמי | נהג + נוסע אחד |
| אחורי | 3 מקומות |

ברכב 5 מקומות:

- עם **הורה אחד**: אפשר לבחור עד ארבעה ילדים, ואחד מהם ישובץ במושב הקדמי לפי התור או לפי מקום קבוע.
- עם **שני הורים**: אפשר לבחור עד שלושה ילדים, וכל הילדים ישובצו מאחור.
- נסיעות עם שני הורים אינן משפיעות על תור המושב הקדמי, כי אף ילד לא יושב מקדימה.
- המקומות הקבועים של רכב 5 מקומות נשמרים בנפרד מהרכב המשפחתי.

## 📱 התקנה על אנדרואיד

1. פתח את [האפליקציה](https://shmuelse.github.io/carseats-pwa/) ב-Chrome.
2. לחץ על **"התקן כאפליקציה"** אם הכפתור מופיע, או השתמש בתפריט Chrome:  
   שלוש נקודות ⋮ ← **הוסף למסך הבית** / **Install app**.
3. לאחר ההתקנה האפליקציה תופיע עם אייקון במסך הבית.
4. אחרי ההתקנה כפתור ההתקנה אמור להיעלם.

> הערה: Chrome מציג את אפשרות ההתקנה רק כשהאתר עומד בתנאי PWA, למשל Service Worker פעיל, Manifest תקין והאפליקציה עדיין לא מותקנת.

## 🛠️ טכנולוגיות

- HTML / CSS / JavaScript — ללא frameworks.
- PWA עם Service Worker לתמיכה Offline.
- Manifest להתקנה כאפליקציה.
- `localStorage` לשמירת נתונים מקומית.
- GitHub Pages לאחסון.
